# SONAR Rock vs Mine Prediction

This project uses Logistic Regression to classify underwater objects as Rock or Mine based on SONAR signal data.

## Technologies Used
- Python
- NumPy
- Pandas
- Scikit-Learn
- Google Colab

## Machine Learning Algorithm
- Logistic Regression

## Dataset
- SONAR Dataset
- 208 observations
- 60 feature columns
- 1 target column (Rock or Mine)

## Project Workflow
1. Data Loading
2. Data Exploration
3. Feature and Label Separation
4. Train-Test Split
5. Logistic Regression Model Training
6. Accuracy Evaluation
7. Predictive System Development

## Results
- Training Accuracy: 83.42%
- Testing Accuracy: 76.19%
